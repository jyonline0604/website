## 有聲畫小説生成規則

### 重要教訓
- 每章音頻必須獨立生成，不能複製或覆蓋
- 每章 HTML 必須獨立創建，不能複製舊章節
- 場景圖片必須根據該章節**實際內容**生成

### 生成流程
1. 提取章節實際內容
2. 根據內容生成符合劇情的場景描述
3. 分開生成 3 張圖片（MiniMax Image-01）
4. 生成音頻（先 MiniMax TTS，失敗則用 Edge TTS）
5. 獨立創建 HTML 頁面
6. 更新 av-novels.html 目錄

### 音頻檔案命名
- 格式：chapter{N}_audio.mp3
- 例如：chapter14_audio.mp3
- 不再使用：chapter{N}_edge_tts.mp3

### 場景圖片數量
- 根據該章節劇情發展需要
- 可能 2-4 張或更多
- 關鍵場景需要多張圖片說明時就增加

### 配額預算（重要！）
- MiniMax Image-01：每天 50 張
- 估算：每章約 3-4 張（視劇情複雜度）
- 建議：每天製作 1-2 章，留 10+ 張配額應急
- 追蹤：定期檢查已用數量

### MiniMax 配額使用（2026-03-31）
